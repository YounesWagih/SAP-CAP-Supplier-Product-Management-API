# Junior SAP CAP Developer Task — Supplier Product Management API

**Technology Stack:** Node.js / TypeScript, SAP CAP, SQLite

**Goal:** Build a fully functional Supplier Product Management API. The API should allow users to manage products and automatically enrich them with external data from a public API.

---

## 1. Entities (CDS Schema)

* **Supplier**

  * Fields: ID, name, email, rating (1–5, validated)
* **Product**

  * Fields: ID, name, price, category, externalRating, averageRating, supplier
  * `externalRating` fetched on creation; `averageRating` updated via action
* **ProductReview**

  * Fields: ID, product (FK), rating (1–5), comment, reviewer
  * Created via `submitReview` action

**Associations:**

* Product → Supplier (many-to-one)
* Product → ProductReview (composition)

---

## 2. Services

* Expose a single service named `CatalogService` with full CRUD for:

  * Suppliers
  * Products
  * ProductReviews

---

## 3. Custom Logic / Event Handlers

**On CREATE Product:**

1. Call external API: `https://fakestoreapi.com/products`
2. Find a product matching the new product’s category
3. Copy `rating.rate` into `externalRating`
4. Handle errors gracefully (do not block creation)

**Custom Action — submitReview:**

* Signature: `action submitReview(productID: UUID, rating: Integer, comment: String)`
* Logic:

  1. Create ProductReview linked to `productID`
  2. Recalculate `averageRating` across all reviews
  3. Update Product’s `averageRating`

---

## 4. Validations

* Product price must be > 0
* Supplier rating must be between 1 and 5
* ProductReview rating must be between 1 and 5

---

## 5. External API Integration

* Use `async/await` for API calls
* Handle HTTP errors and network failures gracefully
* Parse JSON response to extract `rating.rate`
* Never block product creation if API is unavailable

---

## 6. Deliverables

* Git repository with full project
* README including:

  * Installation and run instructions
  * Design decisions and trade-offs
  * Assumptions made
  * Sample API calls:

**Sample API calls:**

* Create Product: `POST /catalog/Products { "name": "T-Shirt", "price": 29.99, "category": "clothing", "supplier_ID": "..." }`
* List Products: `GET /catalog/Products`
* Submit Review: `POST /catalog/submitReview { "productID": "...", "rating": 4, "comment": "Good quality" }`
